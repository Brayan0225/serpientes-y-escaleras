# Proyecto: Serpientes y Escaleras

## Descripcion
Juego de Serpientes y Escaleras (Snakes & Ladders) desarrollado en HTML + JS vanilla para correr localmente en navegador.

## Stack
- HTML5 + CSS3
- JavaScript vanilla
- Sin frameworks ni dependencias externas
- Debe correr localmente abriendo el HTML en el navegador

## Juego - Reglas base (de la imagen referencia)
- Tablero de 20 casillas (serpentina/camino curvo)
- Escaleras que avanzan al jugador
- Serpientes que retroceden al jugador
- 2-4 jugadores (fichas: verde, azul, amarillo, rojo)
- Dado de 6 caras
- Casilla Start -> Casilla 20 (Finish)
